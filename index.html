IT INFRASTRUCTURE BRIEFING | March 11, 2026

AZURE / IaaS

Azure Default Outbound Internet Access Removal – End of March 2026

Azure will remove default outbound internet for new virtual networks at the end of March 2026. New VMs and services may fail Windows activation, Intune sync, and updates unless you provision an Azure NAT Gateway or equivalent outbound path.

Update deployment templates and automation now to avoid SNAT exhaustion and connectivity failures.

Source: HubSite365 Azure Update

Hotpatching Now GA for Azure VM Images

Hotpatching is now available for Azure VM images, allowing users to apply patches without restarting their virtual machines.

If you're on Azure Standard or Premium VMs with the latest Servicing Stack, you can apply most of March's 83 patches without reboots. Major uptime win for production workloads.

Source: NT Compatible

Azure IoT Explorer CVEs

Azure IoT Explorer CVEs could let an attacker read telemetry from connected devices.

If you're using Azure IoT Explorer in production, prioritize the March patches.

Source: NT Compatible

Azure Compute Gallery & Arc CVEs

Notable mentions include CVE-2026-26141 in the Hybrid Worker Extension for Arc-enabled Windows VMs and CVE-2026-26117 in the Azure Connected Machine Agent.

These are crucial for maintaining cloud-based environment security integrity.

Source: CyberWebSpider

INTUNE / ENTRA

Microsoft Entra Connect SyncJacking Hardening – Enforcement Begins March 2026

Microsoft has implemented new safeguards to block account takeover attempts via hard match abuse in Microsoft Entra Connect (known as SyncJacking). Enforcement of this change begins in March 2026.

Enforcement logic now checks OnPremisesObjectIdentifier to detect and block remapping attempts.

Upgrade to the latest Entra Connect version and review hardening guidance immediately.

Source: Microsoft Learn

Entra: Hard-Match Restrictions for Privileged Roles – June 2026

Beginning June 1 2026, Microsoft Entra ID will block any attempt by Microsoft Entra Connect Sync or Cloud Sync from hard-matching a new user object from Active Directory to an existing cloud-managed Microsoft Entra ID user object that holds privileged roles.

This prevents attackers from taking over privileged cloud users by manipulating on-prem AD attributes.

Source: Microsoft Learn

Passkey Profiles Auto-Enabled – March-May 2026

Beginning March 2026, Microsoft will automatically enable Passkey Profiles and shift registration campaigns toward passkey enrollment instead of traditional MFA apps. Tenants that have not previously configured passkey profiles may be automatically migrated to the new configuration.

Review your authentication policies now if you have strong opinions about passkey rollout timing.

Source: SSOJet

Authenticator Jailbreak/Root Detection – February 2026 Rollout

Starting February 2026, we'll introduce Jailbreak/Root detection for Microsoft Entra credentials in the Authenticator app. All existing credentials on such devices will be wiped to protect your organization.

Notify end users about this change; Authenticator will become unusable for Microsoft Entra accounts on jailbroken/rooted devices.

Source: Microsoft Learn (Entra)
